리스트 태그
   
   1. 순서 없는 리스트
      <ul>태그 안에 <li>태그 사용
      
      <ul>
         <li>텍스트1</li>
         <li>텍스트2</li>
         <li>텍스트3</li>
      </ul>
   
   2. 순서 있는 리스트
      <ol>태그 안에 <li>태그 사용
      
      <ol>
         <li>텍스트1</li>
         <li>텍스트2</li>
         <li>텍스트3</li>
      </ol>
      
      type 속성
      type 속성의 값: 1, A, a, i, I

<p> 태그, 글 단락 태그
   단락이란, 내용상 끊어서 구분할 수 있는 하나하나의 부분을 의미하고, 문단이라고 부른다.
   <p> 태그를 이용하려면 이러한 단락을 표현한다.

   링크 태그
   
   HTML 링크(Link)
   현재 페이지에서 다른 페이지로 이동하고자 할 때 하이퍼링크를 사용한다.
   보통 링크라고 부르며, <a>태그를 사용한다.
   
   <a href="" target="">링크를 걸어줄 내용</a>
   
   - href: 페이지의 경로, 사이트 주소
   - target: _blank(새탭), _self(지금 탭), _parent(팝업창을 띄워준 탭)

HTML 요소의 종류 
블록 요소
	안에 내용이 어떻든 간에 한줄을 차지
	코드 상에 한 줄로 이어 써도 화면 상에서는 앞 뒤 요소 사이에 새로운 줄을 만들어서 나타난다.
	영역이 정확히 구분되어 있기 때문에 넓이와 높이 속성을 수정할 수 있다.
	margin-top , margin-bottom 속성도 잘 적용된다.
	padding-top , padding-bottom 속성도 가능
	
	** div태그
		1. 다른 HTML요소들을 하나로 묶는 데 사용되는 대표적인 블록 요소이다.
		2. 주로 여러 요소들의 스타일을 한 번에 적용하기 위해 div 태그를 사용한다.
	
인라인 요소
	-새로운 줄을 만들지 않고 작성한 단락 내에 나타난다.
	-안에 있는 내용만큼만 영역을 차지한다.
	-영역이 불분명(컨텐츠 크기만큼 바뀌므로)하기 때문에 넓이와 높이 속성을 부여할 수가 없다.
	margin-top , margin-bottom 속성 적용 불가능
	padding-top , padding-bottom 속성 적용 불가능
	span태그
		1.텍스트의 특정 부분을 묶는데 자주 사용되는 요소
		2.주로 텍스트의 특정 부분에 따로 다른 스타일을 적용하기 위해 사용한다
	
인라인-블록 요소
->div태그로 인라인 요소를 감싸줄 수 없을 때 사용
 - inline 요소와 동일한 영역을 가지지만 넓이와 높이를 설정할 수 있다.
 margin-top , margin-bottom 속성도 잘 적용된다.
	padding-top , padding-bottom 속성도 가능
	

********인라인 블록 요소는 엔터나 공백을 공백으로 인식한다!!!!************
********만약 가독성 목적으로 요소 별 엔터를 삽입한다면, inline 요소인지 확인해야한다.******
